Формула и объяснение

Базовая формула

PriorityScore = w1*T_wait + w2*C_value + w3*U_comm + w4*P_sale + w5*R_churn

Текущая UI-конфигурация весов

PriorityScore = 0.25*T_wait + 0.30*C_value + 0.20*U_comm + 0.15*P_sale + 0.10*R_churn

Почему клиент в фокусе

Факторы расчета

T_wait

Срочность по времени ожидания ответа. На следующем этапе будет считаться по диапазонам.

C_value

Ценность клиента: портфель, активность и общий бизнес-потенциал.

U_comm

Срочность и характер текущей коммуникации: urgency, tone, responsiveness.

P_sale

Склонность к покупке: product fit, свободный кэш, история отклика и gap в портфеле.

R_churn

Риск потери клиента. Пока только визуальный блок для UX, без backend-расчета.
